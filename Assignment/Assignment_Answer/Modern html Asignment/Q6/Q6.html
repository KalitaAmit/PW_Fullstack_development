Difference between HTML and HTML5
HTML HTML5
  1. Uses older, non-semantic elements like <div> and <span> for layout.
  1. Introduces semantic elements like <header>, <nav>,<section>, and <footer> for better content structure.
  1. No native support for video and audio embedding.
  1. Supports <video> and <audio> elements for embedding media without plugins likeFlash.
  1. Does not support the canvas element for drawing graphics.
  1. Introduces the <canvas> element for rendering 2D graphics directly in thebrowser.
