Question - Explain the use of grid-auto-row and grid-auto-column using code examples

Ans: - 

In CSS Grid Layout, the grid-auto-rows and grid-auto-columns properties control the size of implicitly created rows or columns in the 
grid. These come into play when we defined grid template doesn't have enough rows or columns to accommodate all the content.

grid-auto-row - 

This property specifies the size of implicitly created grid row tracks. It allows us to define a fixed size, a minimum size, a maximum 
size, or a flexible size (using fr units).


syntex of grid-auto-row- 

grid-auto-rows: auto;

grid-auto-rows: 50px;

grid-auto-rows: min-content;

grid-auto-rows: minmax(30px, auto);


code example - 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>grid-auto-rows Example</title>
    <style>
        .container {
            display: grid;
            grid-template-rows: 100px;
            grid-template-columns: 100px 100px;
            grid-auto-rows: 50px; /* Implicit rows will have a height of 50px */
            gap: 10px;
            border: 2px solid black;
        }

        .item {
            background-color: lightblue;
        }
    </style>
</head>
<body>
    <div class="container">
        <div class="item">1</div>
        <div class="item">2</div>
        <div class="item">3</div>
        <div class="item">4</div>
    </div>
</body>
</html>





grid-auto-column - 

The grid-auto-columns property is used to define the width of implicitly created columns in a grid layout. Implicit columns are 
automatically generated when a grid item is placed outside the range of explicitly defined columns.


syntex of grid-auto-column - 

grid-auto-columns: auto;

grid-auto-columns: 50px;

grid-auto-columns: min-content;

grid-auto-columns: minmax(30px, auto);


code example - 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>grid-auto-columns Example</title>
    <style>
        .container {
            display: grid;
            grid-template-rows: 100px 100px;
            grid-template-columns: 100px;
            grid-auto-columns: 50px; /* Implicit columns will have a width of 50px */
            gap: 10px;
         
        }

        .item {
            background-color: lightcoral;
        }

    </style>
</head>
<body>
    <div class="container">
        <div class="item">1</div>
        <div class="item">2</div>
        <div class="item span">3</div>
    </div>
</body>
</html>
