1. title - Sets the title of the web page, which appears in the browser tab.
  2. body - Contains the content that is visible on the web page, including text, images, and interactive elements.
  3. h1 to h6 - Heading tags used to define headings on a page, with <h1> being the most important and h6 the least important.
  4. p - Defines a paragraph of text.
  5. a - Defines a hyperlink, allowing users to navigate to another page or resource.
  6. img - Embeds an image into the web page. It uses the "src" attribute to specify the image URL.
  7. ul - Defines an unordered list of items, typically rendered as bullet points.
  8. ol - Defines an ordered list of items, typically rendered with numbers.
  9. li - Defines a list item within an ordered ol or unordered ul list.
  10. div - A generic container for grouping HTML elements, often used for layout purposes.
  11. span - A generic inline container used to group text or elements for styling purposes without breaking the flow of content.
  12. form - Defines a form used to collect user input, often with input, textarea>, and button elements.
  13. input - Defines an input field for forms, such as text fields, checkboxes, or radio buttons.
  14. button - Defines a clickable button, often used to submit forms or trigger events.
  15. table - Defines a table structure for displaying tabular data, often used with tr, th, and td tags.
