Q4. What is the purpose of the <meta> tag in HTML? How is it commonly used?

Ans:- 

Purpose of the <meta> Tag in HTML
The <meta> tag provides metadata (information about the webpage) to browsers, search engines, and other web services. It helps with SEO,
character encoding, responsiveness, and page behavior. The <meta> tag is placed inside the <head> section of an HTML document.

Common Uses of the <meta> Tag :

(1) setting Character Encoding - Ensures proper text rendering, especially for special characters.

<meta charset="UTF-8">
Defining Viewport for Responsive Design

(2) Makes the website mobile-friendly by controlling how the page scales on different devices.

<meta name="viewport" content="width=device-width, initial-scale=1.0">
Search Engine Optimization (SEO)

(3) Provides a description for search engines to display in search results.

<meta name="description" content="Learn web development with HTML, CSS, and JavaScript.">

(4) specifies relevant keywords for search engines (not commonly used anymore).

<meta name="keywords" content="HTML, CSS, JavaScript, Web Development">
Controlling Page Refresh and Redirection

(5) Automatically refreshes or redirects the page after a set time.

<meta http-equiv="refresh" content="5; url=https://example.com">
Setting Author and Copyright Information

(6) Defines the webpage author.

<meta name="author" content="John Doe">
Preventing Search Engine Indexing

(7) Tell search engines not to index the page.

<meta name="robots" content="noindex, nofollow">




Example of a <meta> Tag in an HTML Document

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="A complete guide to HTML meta tags for SEO and web development.">
    <meta name="author" content="Amit">
    <title>Meta Tags in HTML</title>
</head>
<body>
    <h1>Understanding Meta Tags in HTML</h1>
</body>
</html>
